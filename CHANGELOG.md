### Added
- Project initialized under the name **tempify** (temporal densification for raster stacks).
- Initial project structure following Spec-Driven Development + Harness Engineering.
- Six initial specs in draft state:
  - `core-interpolation`
  - `structure-detection`
  - `temporal-frequency-resolver`
  - `io-handlers`
  - `validation`
  - `cli`
- Steering documents: product, tech, architecture, conventions, harness rules.
- Slash commands: `/spec-init`, `/spec-design`, `/spec-tasks`, `/impl`, `/review`.
- Git hooks: pre-commit, pre-task, post-task.
- Empirical validation reference: Quinta Normal 2020 experiment + 3x3 synthetic stack.
